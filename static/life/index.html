(shoddy) Conway's Game of Life

While the simulation is stopped, you can click (or drag) on a cell to toggle it on or off. Features (and non-features):

  • There are "walls", the grid size is finite with no wrap-around.
  • For your viewing pleasure there is a color gradient on live cells.
  • You can change the tick time with the slider to modify the simulation speed.
